# Coach Mode Prompt

Use this when students need explanation before implementation.

```text
Switch to Coach Mode.

Explain in simple language:
1. What you understand about our idea.
2. What the user problem is.
3. Whether the AI role is structurally necessary.
4. What the smallest credible prototype should show.
5. Which files would probably need to change.
6. What you recommend doing first.
7. What the expected result will be.

Do not edit files yet.
Ask us to approve the first build step.
```
