# Repository Files Each Team Should Have

Minimum required files:

- `README.md`
- `OPENCLAW_INSTRUCTIONS.md`
- `OPENCLAW_BUILD_LOG.md`
- `DEPLOYMENT_STATUS.md`
- `AI_USE_DISCLOSURE.md`
- `TEAM_LOG.md`
- `DECISIONS.md`
- `.env.example`
- `.gitignore`

Optional but useful:

- `PROTOTYPE_CHECKLIST.md`
- `GOVERNANCE_AND_RISK.md`
- `FINAL_EVIDENCE.md`

Core app structure for the default stack:

```text
app/
  page.tsx
  layout.tsx
  globals.css
  api/
    health/
      route.ts
components/
lib/
public/
package.json
tsconfig.json
next.config.ts
tailwind.config.ts
```
